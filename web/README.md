finmgr-web
===

[![CircleCI](https://circleci.com/gh/ksean/finmgr.svg?style=svg)](https://circleci.com/gh/ksean/finmgr)

A simple web interface implementation of the finmgr-core API project.

## Requirements

yarn >= 4.4.1

## Steps

`yarn`

`yarn start`
