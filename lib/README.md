finmgr-lib
===

[![CircleCI](https://circleci.com/gh/ksean/finmgr.svg?style=svg)](https://circleci.com/gh/ksean/finmgr)

A library for defining financial transactions, and providing built-in financial functions to operate on them.

## Requirements

Java >= 21

## Testing

`mvn test`
